26. i wanna know about frasa??


Frasa di bahasa inggris yang saya ketahui ada 5:

1. Noun (Kata benda) 
-> Contoh: food, park, book, opinion, inspiration.

2. Verb (Kata kerja)
-> Contoh: walking, writing, going, typing, creating.

3. Adjective (Kata sifat)
-> Contoh: Beautiful, attractive, large, clever, kind.

4. Adverb (Kata keterangan)
-> Contoh: ...tomorrow, ....yesterday, ....strongly, ....quitely, ....... slowly.

5. Prepositional (Kata depan)
-> Contoh: In..., at....., after....., before....., on......, under.....

35. Apa rumus dari past perfect tense yang benar?Help me!I wanna know.​


Jawaban:

Past perfect tense nomina :

Positive : subjek + had + been + pelengkap + ...

Negative : subjek + had not + been + pelengkap + ...

Interrogative : had + subjek + been + pelengkap + ...

Past Perfect Tense verba :

Positive : Subjek + had + verb 3 + objek + ket.waktu

Negative : subjek + had not + been + objek + Ket.waktu

Interrogative : had + subjek + verb 3 + objek + ket.waktu
