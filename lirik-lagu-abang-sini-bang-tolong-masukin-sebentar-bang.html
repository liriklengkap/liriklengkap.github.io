26. pliss lah bang kak tolong y Abang-Abang Kaka Kaka ​


Jawab:

Pada gambar tersebut, AF // KH, besar sudut ABC = 58° dan sudut BDE = 144°. Besar sudut sudut KGJ = 36⁰ dan besar sudut JGI = 86⁰. Perhatikan gambar pada lampiran: garis a // garis b, dan garis c memotong garis a dan garis b berturut-turut di titik A dan titik B.  

Sudut sehadap (besar sudutnya sama), yaitu

∠A₁ = ∠B₁

∠A₂ = ∠B₂

∠A₃ = ∠B₃

∠A₄ = ∠B₄

Sudut dalam bersebrangan (besar sudutnya sama), yaitu

∠A₂ = ∠B₃

∠A₄ = ∠B₁

Sudut luar bersebrangan (besar sudutnya sama), yaitu

∠A₃ = ∠B₂

∠A₁ = ∠B₄

Sudut dalam sepihak (jumlah sudutnya sama dengan 180⁰)

∠A₂ + ∠B₁ = 180⁰

∠A₄ + ∠B₃ = 180⁰

Sudut luar sepihak (jumlah sudutnya sama dengan 180⁰)

∠A₁ + ∠B₂ = 180⁰

∠A₃ + ∠B₄ = 180⁰
